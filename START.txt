항상 주소만으로 열기(권장)

1) 최초 1회 설치
   npm install
   npm run always:on

2) 이후에는 서버를 따로 켤 필요 없이 주소만 열면 됩니다.
   http://127.0.0.1:8000

설명
- 로그인할 때 자동 시작됩니다.
- 서버가 죽으면 Scheduled Task + loop가 자동 재시작합니다.
- 같은 PC에서만 항상 127.0.0.1:8000 으로 접속합니다.

상태 확인
- npm run always:status

중지/제거
- npm run always:off

수동 실행(일회성)
- npm start

개발 모드(HMR)
- npm run dev
- 브라우저는 터미널에 표시된 Vite URL 사용
